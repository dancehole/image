GitHub 图床管理系统

Image Bed Management System

📋 项目简介

这是一个基于 GitHub 的图床管理系统，利用 GitHub 仓库作为图片存储后端，通过 GitHub API 进行图片的上传、管理和浏览。

系统支持前端图片压缩、水印添加、分类管理等功能，可部署到 GitHub Pages 实现在线访问。

🌟 主要功能

  • 🖼️ 图片浏览（网格/列表视图）
  • 📤 拖拽上传（支持批量）
  • 🗜️ 图片压缩（自定义质量）
  • 🏷️ 水印添加（自定义位置、颜色）
  • 🔍 分类筛选与搜索
  • 📋 上传历史记录
  • 🔗 多种链接格式（URL、Markdown、HTML、BBCode）

🛠️ 技术栈

  • HTML5 / CSS3 / JavaScript
  • jQuery 3.x
  • GitHub REST API
  • jsDelivr CDN
  • GitHub Pages + Actions
